SET 遊戲說明

SET 是一個觀察力與邏輯思考的紙牌遊戲。每張卡片有四個特徵，每個特徵有三種可能:

  • 顏色：紅色、綠色、紫色
  • 形狀：橢圓、波浪、菱形
  • 數量：一個、兩個、三個
  • 填充：實心、條紋、空心

一個有效的SET必須符合以下規則：對於每個特徵，三張卡片要麼全部相同，要麼全部不同。

遊戲規則:

  1. 選擇一位玩家
  2. 找出三張形成SET的卡片
  3. 如找到有效SET，玩家得3分
  4. 如選擇了無效SET，玩家扣1分
  5. 找到SET後，會自動補充新卡片
